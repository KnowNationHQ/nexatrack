# Nexatrack Express Courier Services

A professional courier and logistics website for **Nexatrack Express Courier Services**, serving Florida and the Southeast USA.

## Pages

| Page | Route | Description |
|------|-------|-------------|
| Home | `/` | Hero, stats, services overview, features |
| About | `/about.html` | Company info, team, mission, stats |
| Services | `/service.html` | All courier/logistics service offerings |
| Features | `/feature.html` | Key features and benefits |
| Quote | `/quote.html` | Request a shipping quote |
| Tracking | `/tracking.html` | Real-time package tracking with map |
| Contact | `/contact.html` | Contact form, info, map |

## Features

- **Package Tracking** — real-time tracking with Leaflet map, 3 sample tracking IDs
- **Responsive Design** — Bootstrap 5, works on all devices
- **Animated UI** — Owl Carousel, WOW.js, CSS animations
- **Contact Form** — formspree.io integration (`https://formspree.io/f/meoebapa`)
- **SEO Ready** — Open Graph tags, meta descriptions

## Tech Stack

- HTML5, CSS3, JavaScript
- Bootstrap 5
- Font Awesome 5, Bootstrap Icons
- Leaflet.js (tracking map)
- Owl Carousel
- Animate.css / WOW.js

## Develop

Open any `.html` file in a browser — no build step needed.

## Deploy

Auto-deployed to **Vercel** on push to `main`.

**Live site:** https://nexatrack.vercel.app
